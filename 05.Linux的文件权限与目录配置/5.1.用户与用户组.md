# 5.1 用户与用户组

Linux 是个多人多任务系统，可能会有多人同时使用一台主机来工作。为了让各个用户具有保密的文件数据，因此需要文件权限管理。

## 用户身份

Linux 一般将文件可读写的身份分为三个类别：

- 拥有者（owner）
- 所属群组（group）
- 其他人（others）

## 权限

三种身份各有权限：

- 读（read）
- 写（write）
- 执行（execute）

## 配置文件

Linux 用户身份与用户组记录的文件：

- /etc/passwd：所有的系统上的账号与一般身份用户，以及 root 相关信息
- /etc/shadow：个人密码
- /etc/group：所有的组名
